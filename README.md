# Phaser 3開発例

Phaser 3でゲームを開発してみます。

## 要件

`npm`で依存関係のインストールとスクリプトを実行するためには、[Node.js](https://nodejs.org)が必要です。

## 利用可能なコマンド

| コマンド | 説明 |
|---------|-------------|
| `npm install` | プロジェクトの依存関係をインストールする |
| `npm run dev` | 開発用ウェブサーバーを起動する |
| `npm run build` | `dist`フォルダ内にプロダクションビルドを作成する |
| `npm run dev-nolog` | 匿名データを送信せずに開発用ウェブサーバーを起動する |
| `npm run build-nolog` | 匿名データを送信せずに`dist`フォルダ内にプロダクションビルドを作成する |

## プロジェクトの構造

プロジェクトは、以下のような構造になっています。

| パス                         | 説明                                                     |
|------------------------------|----------------------------------------------------------|
| `index.html`                 | ゲームを格納する基本的なHTMLページ。                        |
| `public/assets`              | ゲームのスプライトやオーディオなど。実行時に直接提供されます。 |
| `public/style.css`           | グローバルなスタイルシートです。                            |
| `src/main.ts`                | アプリケーションのブートストラップ。                        |
| `src/game`                   | ゲームのコードが含まれるフォルダー。                        |
| `src/game/main.ts`           | ゲームのエントリーポイント：ゲームの構成と開始。             |
| `src/game/scenes`            | すべてのPhaserゲームシーンを含むフォルダー。                |
